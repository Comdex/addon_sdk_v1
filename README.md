addon_v1
========

This is wifiIO module's addon SDK repository. 

* addons：文件夹包含应用插件，其中含有代码示例和相应说明。
* addons_hidden：暂未发布程序
* doxygen：doxygen源文件
* gh-pages：github发布页面
* include：头文件
