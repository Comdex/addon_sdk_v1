# wifi.io 轻量开放的云硬件

## 插件（addon）开发SDK V1.0

This is wifiIO module's addon SDK repository. 

* addons：文件夹包含应用插件，其中含有代码示例和相应说明。
* addons_hidden：暂未发布程序
* doxygen：doxygen源文件
* gh-pages：github发布页面
* include：头文件

***
初学者建议阅读顺序：

1. helloworld
2. helloRGB
3. dht11
4. ot_data
5. 这里基本就入门咯，剩下的随便看啦
