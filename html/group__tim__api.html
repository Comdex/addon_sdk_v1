Detailed Description

Only tim6 and tim7 can be used, for precise time caculation.

api_tim.tim6_init(100000); //初始化定时器6 10us为单位 api_tim.tim6_poll(2000); //delay 2000x10us > 18ms

api_tim.tim6_start(10); //start cnt to 100us while(!api_tim.tim6_is_timeout()); //wait
