int(* init)(wifiIO_pwm_t pwm_peripheral, u16_t prescaler, u16_t init_T, u16_t init_t)

PWM init.

PWM generated by Timer whose income clock is 60000000/prescaler.e.g. prescaler:600 init_T:100 will generate 60000000/60000 = 1000Hz PWM, init_t=30 means the duty cycle is 30%

Parameters
[in]init_T: Cycle counted in tick
[in]init_t: High level counted in tick
Returns
STATE_OK : Successful. other : if an error occurred.
